# blinkit-sales-analysis-dashboard
This Power BI project analyzes Blinkit's sales data, covering the entire data lifecycle from cleaning to dashboard creation. It provides insights on revenue trends, outlet efficiency, and customer satisfaction across demographics and timeframes.
Blinkit Sales Analysis Dashboard (Power BI)
Project Overview
This repository contains a comprehensive Power BI project focused on the sales analysis of Blinkit, a prominent quick-commerce service. The project demonstrates the full data lifecycle—from requirement gathering and data cleaning to the creation of a high-fidelity, interactive dashboard. The primary goal is to provide business stakeholders with actionable insights regarding revenue trends, outlet efficiency, and customer satisfaction across various demographics and timeframes.

Key Metrics and Business Insights
The dashboard provides a deep dive into several critical performance indicators:

Sales Performance: Monitoring total revenue, average sales per transaction, and average customer ratings.

Outlet Segmentation: Comparison of performance based on outlet size (Small, Medium, High) and geographic location (Tier 1, Tier 2, and Tier 3 cities).

Establishment Analysis: A historical view of how outlets established in different years contribute to the current sales mix.

Item Distribution: Analysis of product categories to identify top-performing items and inventory trends.

Technical Implementation
The project utilizes the following technical workflows:

ETL Process: Used Power Query to clean and transform raw Excel data, ensuring data integrity and handling null values.

Data Modeling: Built a robust relationship model to ensure seamless filtering across different data dimensions.

DAX Calculations: Implemented Data Analysis Expressions (DAX) to create custom measures for complex KPIs and dynamic reporting.

Advanced Visualization: Use of donut charts, bar charts, and tree maps with integrated tooltips and "Edit Interactions" for precise cross-filtering.

Tech Stack
Software: Power BI Desktop

Data Source: MS Excel

Language: DAX (Data Analysis Expressions)
